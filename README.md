#Hospital Emergency Room Dashboard Project

This project is a Hospital Emergency Room Dashboard created to analyze and visualize patient data in a simple and interactive way. The dashboard helps hospital staff understand patient flow, waiting time, admission status, and overall service quality.

Key Highlights:

Displays total number of patients (513) with a monthly and yearly filter for better analysis.

Tracks average waiting time (36.32 minutes) to monitor emergency room efficiency.

Shows admission status, including admitted and not admitted patients with percentage breakdown.

Analyzes patient demographics such as age groups and gender distribution.

Measures patient satisfaction score (4.96) to evaluate service performance.

Visualizes on-time vs delayed visits and department-wise referrals.


Tools & Skills Used:

Excel (Data Modeling, DAX, Interactive Visuals)

Data Cleaning & Transformation

Dashboard Design & Storytelling


Project Outcome:

This dashboard provides actionable insights that help hospital management improve patient experience, reduce waiting time, and optimize emergency room operations.
